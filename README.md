## Hospital Management System

A simple Hospital Management System (HMS) built using Flask, MySQL, and Bootstrap.
It supports Admin, Doctor, and Patient roles with appointment management.

# Features
- Login system (Admin, Doctor, Patient)
- Patient registration
- Doctor & patient management (Admin)
- Book & manage appointments
- Appointment confirmation by Admin
- Emergency appointment priority
- Admin dashboard (total patients, doctors, appointments)
- Modern UI with responsive design

# Tech Stack
Python (Flask)
MySQL
HTML, Bootstrap 5
Custom CSS
